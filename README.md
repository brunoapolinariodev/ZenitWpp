# ZenitWpp

Plataforma de atendimento ao cliente via WhatsApp com automação por IA, gerenciamento de filas, suporte multimídia e análises em tempo real.

## Estrutura do Projeto

```
ZenitWpp/
├── apps/
│   ├── api/          # ASP.NET Core (C#)
│   ├── web/          # Angular
│   └── ai-service/   # Python + FastAPI
├── docs/             # Documentação
├── docker-compose.yml
└── README.md
```

## Como rodar

```bash
docker-compose up
```
